Modelos Matemáticos

Concepto

Definición Expresión formal, simbólica y abstracta de las relaciones entre los componentes de un fenómeno del mundo real.
Expresión Puede ser una simple ecuación o un conjunto complejo de funciones interconectadas.
Lenguaje Describe sistemas de manera objetiva y sin ambigüedades mediante el lenguaje matemático.

Características Deseables

Parsimonia

Prioriza la simplicidad, evitando el uso innecesario de múltiples parámetros.

Modestia

El modelo no aspira a imitar la realidad a la perfección, sino a resaltar los aspectos de interés.

Exactitud

Reproduce el funcionamiento del sistema con valores similares a los observados en la realidad.

Verificabilidad

Permite comparar los resultados teóricos con datos reales.

Componentes Estructurales

Variables

Características de un fenómeno capaces de cambiar o adquirir distintos valores.

De estado / Niveles → cantidad almacenada Flujos → lo que viaja entre componentes
Parámetros y constantes

Valores fijos que anteceden a las variables en una ecuación; no cambian durante la simulación.

Restricciones y condiciones de contorno

Restricciones: límites lógicos del sistema.

Condiciones de contorno: fuentes y sumideros externos.

Representaciones
Ecuaciones físicas (diferenciales) Ecuaciones empíricas (estadísticas / regresión)

Fases de Modelización

① Identificación

Define límites del problema, escala, componentes y ecuaciones representativas (modelo conceptual).

② Calibración

Determina el valor exacto de los parámetros.

③ Validación

Compara resultados del modelo con valores reales históricos para evaluar el margen de error.

④ Simulación

Proyecta escenarios futuros una vez validado el modelo.

⑤ Análisis de sensibilidad

Altera ligeramente los parámetros para observar la influencia de cada uno sobre el resultado final.

Clasificación por Aleatoriedad

Deterministas

El mismo conjunto de entradas produce siempre el mismo resultado exacto. No incluyen propiedades aleatorias; los valores se asumen conocidos.

Estocásticos

Incluyen generadores de procesos aleatorios. Las salidas pueden variar; se enfocan en probabilidades y rangos de variación esperables.

Clasificación por Base Causal (Cajas)

📦 Caja Negra (Empíricos)

Relaciones netamente estadísticas. Alto poder predictivo, pero no explican el "porqué" ni se basan en leyes universales.

🔲 Caja Blanca (Físicos)

Se rigen por leyes físicas comprobadas (conservación de materia y energía). Requieren alto conocimiento del sistema.

🔳 Caja Gris (Heurísticos)

Opción intermedia: partes empíricas integradas con principios físicos lógicos. Basados en interpretaciones probables.

Clasificación por Espacio, Tiempo y Aplicación

Por Espacio
Agregados → área de estudio como un todo Distribuidos → área dividida en celdillas con datos propios
Por Tiempo
Estáticos → resultado consolidado para un periodo Dinámicos → comportamiento variable a lo largo del tiempo
Por Aplicación
Descriptivos / Simulación → predicen escenarios De control → ajustan variables para mantener estabilidad Optimización → buscan la solución más eficiente
Por Estructura Algebraica
Numéricos / Cualitativos Polinomiales (lineales, cuadráticos) Trigonométricos Exponenciales
